# これなに？

Chromeで(半)自動操作とか特定データ取得とかできるようにしたいやつ

# どうやって使う？

## 導入

Chromeに手動で拡張機能として読み込ませます。
拡張機能(chrome://extensions/)を開き、「パッケージ化されていない拡張機能を読み込む」をクリックし、このファイルがあるフォルダを選択します。
その後、開発者ツールを開いたまま、何かしらのサイトを開けば、恐らく起動を完了したような感じのメッセージが表示されると思います。
また、`Ctrl+Shift`で、なんか表示されます。これは、再度`Ctrl+Shift`を押すと消えます。

## 改造

`run.js`と、`usr/`配下を、各ユーザーが改造するものとして想定しています。(この範囲内の改造であれば、ファイルの増減があっても吸収されるように`manifest.json`を設定してあります。)
構成は、JavaScriptの構文として成立し、入出力が想定されている形式になっていれば、多分大丈夫だと思います。
改造後は、この拡張機能の詳細画面にある「更新」をクリックすることで、変更が反映されるはずです。
